# لوحة تحكم المسلسلات التركية

لوحة تحكم ويب لإدارة تطبيق المسلسلات التركية.

## الميزات

- **إدارة المصادر**: تفعيل/تعطيل المصادر وتغيير الدومينات
- **إدارة المسلسلات**: إضافة روابط من مصادر مختلفة للمسلسلات
- **إدارة الحلقات**: إضافة روابط للحلقات من مصادر متعددة
- **الرسائل**: إرسال إعلانات ورسائل صيانة للمستخدمين
- **الإعدادات**: التحكم في إصدارات التطبيق والتحديثات

## كيفية الاستخدام

### 1. فتح الداشبورد

افتح الملف `dashboard/index.html` في المتصفح، أو من GitHub Pages:
```
https://USERNAME.github.io/turkish-series/dashboard/
```

### 2. إعداد GitHub Token

للتمكن من حفظ التغييرات، تحتاج إلى:

1. اذهب إلى GitHub → Settings → Developer settings → Personal access tokens
2. اضغط "Generate new token (classic)"
3. أعطه اسم مثل "Turkish Series Dashboard"
4. اختر صلاحية `repo`
5. انسخ الـ Token

### 3. إدخال الـ Token في الداشبورد

1. في صفحة "الرئيسية"
2. أدخل الـ Token في حقل "GitHub Token"
3. تأكد من صاحب الـ Repo واسمه
4. اضغط "اختبار الاتصال"

### 4. البدء في الاستخدام

بعد الاتصال الناجح، يمكنك:
- تفعيل/تعطيل المصادر من صفحة "المصادر"
- تغيير دومين أي مصدر
- إضافة مصادر جديدة
- إرسال رسائل للمستخدمين

## ملاحظات أمنية

- الـ Token يتم حفظه في localStorage المتصفح فقط
- لا يتم إرساله لأي سيرفر خارجي
- يُفضل استخدام Token بصلاحيات محدودة
